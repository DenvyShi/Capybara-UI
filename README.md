# 水豚魔獸（Turtle WoW）插件合集

為 **Turtle WoW**（水豚魔獸）1.12 經典版整理的插件包，收錄 **156 個**插件，涵蓋 UI 增強、團隊框架、戰鬥統計、任務輔助、職業專用等全方位需求。插件日期 2026-05-28。

## 檔案說明

| 檔案 | 大小 | 用途 |
|------|------|------|
| `AddOns-20260528.zip` | ~74 MB | 完整插件包，解壓至 `Interface/AddOns/` |
| `turtle-wow.exe` | ~33 MB | Turtle WoW 官方啟動器 / 更新程式 |
| `WoW.exe` | ~4.9 MB | WoW 客戶端執行檔 |

## 安裝方式

1. 下載 `AddOns-20260528.zip`
2. 解壓縮到 WoW 目錄下的 `Interface/AddOns/`
3. 進入遊戲，在角色選擇畫面左下角點選「插件」按鈕，勾選需要的插件
4. 建議勾選「載入過期插件」以確保相容性

## 注意事項

- 本合集為 **Turtle WoW 1.12 經典版**專用，其他版本（燃燒的遠征、巫妖王之怒等）不相容
- 部分插件已內建 `zhCN` / `zhTW` 繁體中文語系，若顯示英文可檢查插件設定中的語言選項
- 如遇到報錯，請先確認已勾選「載入過期插件」，或使用 AddOnOrganizer 排查衝突
